Transformer 模型的参数量主要由**四大部分**组成，计算公式如下：

```text
总参数量 ≈ 词嵌入层 + 注意力层 + 前馈网络层 + 输出层（与词嵌入共享）
```

具体到你的 `ModelConfig`：

| 超参数          | 值                 | 说明             |
| :-------------- | :----------------- | :--------------- |
| `dim` (d_model) | 1024               | 模型隐藏维度     |
| `n_layers`      | 18                 | Transformer 层数 |
| `n_heads`       | 16                 | 注意力头数       |
| `n_kv_heads`    | 8                  | KV 头数（GQA）   |
| `vocab_size`    | 6144               | 词表大小         |
| `max_seq_len`   | 512                | 最大序列长度     |
| `head_dim`      | dim / n_heads = 64 | 每个头的维度     |

### 1.词嵌入层（Token Embedding）

```text
参数量 = vocab_size × dim = 6144 × 1024 = 6,291,456 ≈ 6.3M
```

### 2. 输出层（`LM Head`）

在代码中，`self.tok_embeddings.weight = self.output.weight`，所以**输出层与词嵌入层共享权重**，不增加额外参数。

### 3. 每一层 Transformer 的参数

每层包含 **Attention** 和 **MLP** 两部分：

#### （1）Attention 模块

```python
self.wq = nn.Linear(dim, n_heads * head_dim, bias=False)   # 1024 × 1024 = 1,048,576
self.wk = nn.Linear(dim, n_kv_heads * head_dim, bias=False) # 1024 × 512 = 524,288
self.wv = nn.Linear(dim, n_kv_heads * head_dim, bias=False) # 1024 × 512 = 524,288
self.wo = nn.Linear(n_heads * head_dim, dim, bias=False)    # 1024 × 1024 = 1,048,576
```

**Attention 参数量** = 1,048,576 + 524,288 + 524,288 + 1,048,576 = **3,145,728 ≈ 3.15M**

#### （2）MLP 模块

```python
hidden_dim = int(2 * 4 * dim / 3) = int(2 × 4096 / 3) = 2730 ≈ 2730
hidden_dim = 64 × ((2730 + 63) // 64) = 64 × 43 = 2752  # 对齐到 multiple_of=64

self.w1 = nn.Linear(dim, hidden_dim, bias=False)   # 1024 × 2752 = 2,818,048
self.w2 = nn.Linear(hidden_dim, dim, bias=False)   # 2752 × 1024 = 2,818,048
self.w3 = nn.Linear(dim, hidden_dim, bias=False)   # 1024 × 2752 = 2,818,048
```

**MLP 参数量** = 2,818,048 × 3 = **8,454,144 ≈ 8.45M**

#### （3）RMSNorm（两层，每层一个可学习参数）



```python
self.attention_norm = RMSNorm(dim)  # 1024 个参数
self.ffn_norm = RMSNorm(dim)        # 1024 个参数
```

**Norm 参数量** = 2 × 1024 = **2,048**（可忽略不计）

#### （4）单层总参数量

```text
单层 = 3.15M + 8.45M + 0.002M ≈ 11.60M
```

### 4. 所有层叠加

```text
所有层 = 11.60M × 18 ≈ 208.8M
```

### 5. 总参数量

```text
总参数量 = 词嵌入层 + 所有层 + 最终 Norm
        = 6.29M + 208.8M + 0.001M
        ≈ 215.1M
```

这与 `print(f'LLM总参数量：{count_parameters(model) / 1e6:.3f} 百万')` 输出的 **215.127 百万** 完全吻合。



## qwen2.5

```text
{
  "hidden_size": 1536,
  "num_hidden_layers": 28,
  "num_attention_heads": 12,
  "num_key_value_heads": 2,
  "intermediate_size": 8960,
  "vocab_size": 151936,
  "tie_word_embeddings": true
}
```



### 1. 词嵌入层（Token Embedding）



```python
嵌入参数量 = vocab_size × hidden_size
           = 151936 × 1536
           = 233,373,696
           ≈ 2.33 亿
```



### 2. 输出层（LM Head）

由于 `tie_word_embeddings: true`，**输出层与词嵌入层共享权重**，所以这部分不增加额外参数。

否则，会额外增加约 2.33 亿参数，但 Qwen2.5 通过共享节省了这部分开销。

------

### 3. 每一层 Transformer 的参数

每层包含 **Attention** 和 **MLP** 两部分：

#### （1）Attention 模块



```text
head_dim = hidden_size / num_attention_heads = 1536 / 12 = 128

Q 投影: hidden_size × (num_heads × head_dim) = 1536 × 1536 = 2,359,296
K 投影: hidden_size × (num_kv_heads × head_dim) = 1536 × (2 × 128) = 1536 × 256 = 393,216
V 投影: hidden_size × (num_kv_heads × head_dim) = 1536 × 256 = 393,216
O 投影: (num_heads × head_dim) × hidden_size = 1536 × 1536 = 2,359,296

Attention 参数量 = 2,359,296 + 393,216 + 393,216 + 2,359,296
                   = 5,505,024
                   ≈ 5.51M
```



#### （2）MLP 模块（使用 SwiGLU）



```python
# 使用 SiLU (SwiGLU)，需要 3 个线性层
w1: hidden_size × intermediate_size = 1536 × 8960 = 13,762,560
w2: intermediate_size × hidden_size = 8960 × 1536 = 13,762,560
w3: hidden_size × intermediate_size = 1536 × 8960 = 13,762,560

MLP 参数量 = 13,762,560 × 3 = 41,287,680 ≈ 41.29M
```



#### （3）RMSNorm 层（两个）



```python
attention_norm: hidden_size = 1536
ffn_norm: hidden_size = 1536

Norm 参数量 = 1,536 + 1,536 = 3,072 ≈ 0.003M
```



#### （4）单层总参数量



```text
单层参数量 = 5.51M + 41.29M + 0.003M ≈ 46.80M
```



### 4. 所有层叠加 + 最终输出层



```python
所有层 = 46.80M × 28 = 1,310.4M = 13.10 亿

最终 RMSNorm: hidden_size = 1536 ≈ 0.002M

总参数量 = 嵌入层 + 所有层 + 最终 Norm
         = 233.37M + 1,310.40M + 0.002M
         = 1,543.77M
         ≈ 15.4 亿
```



所以，这个配置对应的模型是 **Qwen2.5-1.5B**（约 15.4 亿参数）。
